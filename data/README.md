# Kaggle Dataset Placement

Download Kaggle dataset `uciml/faulty-steel-plates` and place the CSV file here as:

- `data/faults.csv`

If Kaggle provides a different file name, rename it to `faults.csv`.

Expected target format:
- Preferred: one-hot class columns
  - `Pastry`, `Z_Scratch`, `K_Scatch`, `Stains`, `Dirtiness`, `Bumps`, `Other_Faults`
- Alternate: a single `target` or `class` column
